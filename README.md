# 100ExToLearnRust

Requirements : 
- Rust / workshop-runner

Installation (on WSL) :
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
cargo install --locked workshop-runner
```

- Run the workshop-runner to check if an exercice passed :
```
wr
```
